# AuraQR

AuraQR is a modern, responsive QR Code Generator and Scanner web app built with HTML, Tailwind CSS, and JavaScript. It allows users to create stylish QR codes instantly, scan QR codes using camera or image upload, manage history, and generate batch QR codes with a premium UI experience.

## Live Demo

🌐 Live Site: [https://yourusername.github.io/auraqr/](https://shankar2006458.github.io/AuraQr/)

## Features

* Generate QR Codes instantly
* Multiple QR Types:

  * URL
  * Text
  * WiFi
  * Contact (vCard)
  * WhatsApp
  * Email
  * SMS
  * Call
  * UPI Payment
  * Event
* Download QR as PNG, JPG, SVG
* Copy QR Image
* Print QR Code
* Share QR Code
* QR Scanner using:

  * Mobile Camera
  * Image Upload
* Animated Laser Scanner UI
* QR History with Reopen/Delete
* Favorites System
* Batch QR Generator
* Smooth Animations
* Mobile First Responsive Design
* Premium Glassmorphism UI

## Tech Stack

* HTML5
* Tailwind CSS
* Vanilla JavaScript
* QRCode.js
* jsQR
* Lucide Icons
